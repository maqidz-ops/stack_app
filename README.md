# Stack Mobile Apps
Stack is a smart subscription tracker app that helps users manage recurring payments, monitor upcoming renewals, and gain better control over their monthly spending.

## Features
- User Authentication (Login)
- Dashboard Subscription
- Local Notification
- REST API Integration
- State Management
- Local Storage

## Installation
Clone this repository:
```bash
git clone https://github.com/maqidz-ops/stack_app.git
cd stack_app
```

Install dependecies:
```bash
flutter pub get
```

Run the application:
```bash
flutter run
```

## Tech Stack
- Flutter
- Dart
- Provider
- MockAPI
- Shared Preferences
- Flutter Local Notifications
- Local Storage
